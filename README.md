### Important
It takes about 65 seconds for Cassandra node to start, so <i>run-cassandra.sh</i> will run about <b>1 minute</b>.

### Usage
````
$ sudo chmod +x run-cassandra.sh
$ sudo chmod +x shutdown-cassandra.sh
````
